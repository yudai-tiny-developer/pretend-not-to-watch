# Remove from watch history button for YouTube Player

A Chrome extension that adds a **Remove from watch history** button to YouTube, letting you remove the currently open video from your watch history without leaving the player page.

It works on regular YouTube videos and YouTube Shorts.

## Usage

1. Open a video or Short on YouTube.
2. Click **Remove from watch history**.
3. Wait for the button state to change:
   - `Removed` when successful
   - `No items to delete` if no matching history item was found
   - `Failed to remove` if the request did not complete successfully
